Examples of the CPLEX for MATLAB Toolbox

These examples illustrate the usage of toolbox functions to solve mathematical programming problems.
In this section
  • Example: runtoolboxexamples.m
    • This function runs all the toolbox examples in succession.
  • Example: cplexbilpex.m
    • This example use the function cplexbilp to solve a binary integer programming model.
  • Example: cplexlpex.m
    • This example uses the function cplexlp to solve a linear programming model.
  • Example: cplexlsqbilpex.m
    • This example uses the function cplexlsqbilp to solve a constrained least squares problem.
  • Example: cplexlsqlinex.m
    • This example uses the function cplexlsqlin to solve a constrained least squares programming model.
  • Example: cplexlsqmilpex.m
    • Use the function cplexlsqmilp to solve a constrained least squares problem.
  • Example: cplexlsqmiqcpex.m
    • This example uses the function cplexlsqmiqcp to solve a constrained least squares problem.
  • Example: cplexlsqnonneglinex.m
    • This example uses the function cplexlsqnoneglin to solve a nonegative least squares programming model.
  • Example: cplexlsqnonnegmilpex.m
    • This example uses the function cplsqnonnegmilexlp to solve a nonnegative least squares problem with binary variables.
  • Example: cplexlsqnonnegmiqcpex.m
    • This example uses the function cplexlexlsqnonnegmiqcp to solve a nonnegative least squares problem with binary variables and a quadratic constraint.
  • Example: cplexlsqnonnegqcpex.m
    • This example uses the function cplexlsqnonegqcp to solve a nonnegative constrained least squares problem with some quadratic constraints.
  • Example: cplexlsqqcpex.m
    • This example uses the function cplexlexlsqqcp to solve a quadratically constrained least squares problem.
  • Example: cplexmilpex.m
    • This example uses the function cplexmilp to solve a mixed-integer linear programming model.
  • Example: cplexmiqpex.m
    • This example uses the function cplexmiqp to solve a mixed-integer quadratic programming model.
  • Example: cplexqcpex.m
    • This example uses the function cplexqcp to solve a quadratically constrained quadratic programming model.
  • Example: cplexqpex.m
    • This example uses the function cplexqp to solve a quadratically constrained quadratic programming model.
