Example: cplexqpex.m

The QP problem solved in this example is:
    Maximize  x1 + 2 x2 + 3 x3
              - 0.5 ( 33x1*x1 + 22*x2*x2 + 11*x3*x3 - 12*x1*x2 - 23*x2*x3 )
    Subject To
       - x1 +   x2 + x3 <= 20
         x1 - 3 x2 + x3 <= 30
    Bounds
         0 <= x1 <= 40
         0 <= x2
         0 <= x3
See also the example file cplexqpex.m.
