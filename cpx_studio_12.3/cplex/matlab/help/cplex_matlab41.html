Example: lpex1.m

This example illustrates using the Cplex Class API to populate data in a model.
This problem demonstrates different methods for creating a problem. The user has to choose the method via an argument to the function: lpex1('r') generates the problem by adding rows, lpex1('c') generates the problem by adding columns, lpex1('a') generates the problem by adding arrays.
The LP problem solved in this example is:
    Maximize  x1 + 2 x2 + 3 x3
    Subject to
       - x1 +   x2 + x3 <= 20
         x1 - 3 x2 + x3 <= 30
    Bounds
         0 <= x1 <= 40
         0 <= x2
         0 <= x3
See also the example file lpex1.m.
