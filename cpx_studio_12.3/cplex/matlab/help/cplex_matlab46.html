Example: mipex1.m

This example illustrates using the Cplex Class API to enter and optimize a mixed integer programming problem, demonstrating different methods for creating a problem.
The user has to choose the method via an argument: mipex1('r') generates the problem by adding rows, mipex1('c') generates the problem by adding columns, and mipex1('a') generates the problem by adding arrays.
The MIP problem solved in this example is:
    Maximize  x1 + 2 x2 + 3 x3 + x4
    Subject to
       - x1 +   x2 + x3 + 10 x4 <= 20
         x1 - 3 x2 + x3         <= 30
                x2      - 3.5x4  = 0
    Bounds
         0 <= x1 <= 40
         0 <= x2
         0 <= x3
         2 <= x4 <= 3
    Integers
        x4
See also the example file mipex1.m.
