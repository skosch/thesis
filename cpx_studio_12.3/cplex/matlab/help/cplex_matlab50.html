Example: miqpex1.m

This example illustrates using the Cplex Class API to create and solve a MIP with a quadratic objective (MIQP).
This function fills in the data structures for the mixed integer quadratic program:
        Maximize
         obj: x1 + 2 x2 + 3 x3 + x4
              - 0.5 ( 33x1*x1 + 22*x2*x2 + 11*x3*x3
                     -  12*x1*x2 - 23*x2*x3 )
        Subject To
         c1: - x1 + x2 + x3 + 10x4  <= 20
         c2: x1 - 3 x2 + x3         <= 30
         c3:       x2       - 3.5x4  = 0
        Bounds
         0 <= x1 <= 40
         0 <= x2
         0 <= x3
         0 <= x4 <= 3
        Integers
          x4
        End
See also the example file miqpex1.m.
