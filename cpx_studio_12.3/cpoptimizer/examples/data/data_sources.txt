Several CP Optimizer examples use data taken from the academic
literature.  The sources of the data are listed here.  For examples
where the data is embedded in the the code itself and not read from
a file, the relevant citation is given in the source of the example.

goalprog_8_40_0.6_3.data
	R. Sadykov and L. Wolsey.  Integer programming and constraint
	programming in solving a multi-machine scheduling problem with
	deadlines and release dates.  INFORMS Journal on
	Computing 18(2):209-217, 2006.
