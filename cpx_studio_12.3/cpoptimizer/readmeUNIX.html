IBM ILOG CPLEX Optimization Studio 12.3 (CP Optimizer component)

This is the CP Optimizer 12.3 distribution, including Concert Technology. Please read the CP Optimizer 12.3 Release Notes for news on this version of CP Optimizer.

Directory Structure

The following sub-directories (and this file), are contained in the directory where you installed CP Optimizer 12.3.

In the cpoptimizer directory:

  bin/<machine>:
        CP Optimizer shared libraries for Java (through JNI).  The name
        of the shared library is libcp_wrap_cpp_java123.so.

  examples/src/cpp:
	Source files for the C++ examples.

  examples/src/java:
	Source files for the Java examples.

  examples/<machine>/<libformat>:
	Platform-specific makefile for building and executing examples.

  include/ilcp:
	Header files, such as cp.h and cpext.h,
        that are needed to compile C++ applications.

  lib:
	The ILOG.CP.jar file containing the Java classes
        needed to use the Java version of CP Optimizer.

  lib/<machine>/<libformat>:
	CP Optimizer library file for C++ users (libcp.a)

In the concert directory:

  include/ilconcert:
	Header files, such as ilomodel.h,
        that are needed to compile C++ applications.

  lib/<machine>/<libformat>:
	Concert Technology libraries for C++ users (libconcert.a).

Compiling and Executing examples

To compile and execute examples, do the following:
	% cd examples/<machine>/<libformat>
	% make color
to compile a simple C++ CP Optimizer example

A list of all the available examples can be found in the file examples/<machine>/<libformat>/makefile: or you can view the files listed in examples/src/cpp and examples/src/java.
The makefile also contains recommended C++ compiler flags and other settings for building applications on your particular computer and shows how to set the locations of the CP Optimizer JNI shared library and the ILOG.CP.jar file used from Java. 

In the directory examples/<machine>/<libformat> you can do:
	% make all		to compile all examples
	% make execute		to compile and execute all examples
	% make execute_cpp	to compile and execute only the C++ examples
	% make execute_java	to compile and execute only the Java examples
	% make clean		to remove executable files when you have finished

Note that to build and run the Java examples, you must first set your environment variable JAVA_HOME to point to your installation of JDK.
