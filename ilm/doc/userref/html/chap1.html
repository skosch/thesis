Getting Started Quickly

ILOG products are protected by the ILOG License Manager (ILM). Before you can use the product you have installed, you need to set up ILM access keys. This chapter explains how to do this step-by-step for different platforms. More specifically:

  • See the section "Installing Access Keys on Unix" for instructions on how to install ILM access keys on Unix.

  • See the section "Installing Access Keys on Windows" for instructions on how to install ILM access keys on Windows.

  • If you encounter problems when installing or using access keys, go to the section "Troubleshooting" .
