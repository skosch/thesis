Installation Directory

We assume that your ILOG product and ILM have already been installed on your system. If it is not the case, follow the instructions located in the CD-ROM jacket or on your specific installation sheet.

For convenience, we will assume that the ILOG installation directory is /usr/ilog. If this is not the case, please replace subsequent references to /usr/ilog in this document with the actual installation directory.
