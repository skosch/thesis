Installation Directory

We assume that the product and ILM have already been installed on your system. If it is not the case, follow the instructions located in the CD-ROM booklet or on your specific installation sheet.

For convenience, we will assume that ILM is installed in the directory C:\ILOG\ILM. If this is not the case, please replace subsequent references to C:\ILOG\ILM in this document with the actual installation directory.
