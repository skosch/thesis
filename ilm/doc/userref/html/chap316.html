The ihostid Command

Syntax: ihostid

This command displays the current host's hostname, hostid, and hostid checksum.

The hostid of a machine changes when its hard disk is reformatted. This invalidates any previous licenses. If you reformat the hard disk of your machine, you need to get its new hostid and contact ILOG to get new licenses for your products.
