The ilmcheck Command

Syntax: ilmcheck [ -v ] filename ...

This command checks whether the license files given as arguments are correct. Without arguments, ilmcheck checks the default file /usr/ilog/ilm/access.ilm or the files specified by ILOG_LICENSE_FILE if this variable is defined.

The ilmcheck command detects syntax errors, expired keys, duplicate lines, incorrect hostids, and so on.

With the -v option, ilmcheck prints the ILM version number, and shows the checked licenses.
